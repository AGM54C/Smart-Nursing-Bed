## 🔌 接线指南

### ESP32-S3 接线

```
ESP32-S3-N16R8 (USB-C)
┌────────────────────────────────────────┐
│                                        │
│  GPIO2  (SDA) ──→ MAX30102 SDA         │
│                ──→ MLX90614 SDA        │
│                ──→ MPU6050 SDA         │  ← I2C共用总线
│  GPIO1  (SCL) ──→ MAX30102 SCL         │     (面包板同行并联)
│                ──→ MLX90614 SCL        │
│                ──→ MPU6050 SCL         │
│                                        │
│  GPIO4  (ADC1) ──→ 压力矩阵MUX输出    │  ← ADC1通道
│  GPIO5  (ADC1) ──→ HK-2000B+ 信号     │     (WiFi时可用)
│  GPIO6  (ADC1) ──→ 电池分压器中点      │
│                                        │
│  GPIO39,40,41  ──→ MUX1(行) S0,S1,S2  │
│  GPIO42,21,47  ──→ MUX2(列) S0,S1,S2  │
│  GPIO48        ──→ MUX1 SIG(行驱动)    │
│                                        │
│  预留UART1 (正式连续血压传感器):        │
│  GPIO16 (RX)   ──→ 血压传感器TX        │
│  GPIO15 (TX)   ──→ 血压传感器RX        │
│                                        │
│  3.3V ──→ 各传感器 VCC (面包板并联)    │
│  GND  ──→ 各传感器 GND (面包板并联)    │
│                                        │
│  USB-C ──→ 电脑 (烧录+串口调试)        │
└────────────────────────────────────────┘
```

> ⚡ **ESP32-S3优势**：USB-C直接做串口，插电脑就能烧录+调试，不需要额外USB转串口线。

**I2C地址说明**（三个传感器地址不冲突，一条总线即可）：
- MAX30102: `0x57`
- MLX90614: `0x5A`  
- MPU6050:  `0x68`

### Velostat 压力矩阵制作

```
步骤1: 准备基底
  - 裁切30×30cm泡沫垫作为基底

步骤2: 贴行电极
  - 在基底上平行贴8条铜箔条
  - 间距约3cm，总宽度约24cm
  - 一端引出杜邦线，连接到MUX1的8个通道

步骤3: 覆盖Velostat
  - 整张Velostat覆盖在铜箔上
  - 确保完全覆盖所有铜箔条

步骤4: 贴列电极
  - 在Velostat上方垂直贴8条铜箔条
  - 与行电极正交
  - 一端引出杜邦线，连接到MUX2的8个通道

步骤5: 覆盖保护层
  - 用布料或无纺布覆盖保护

电路: 行MUX的COM引脚接3.3V (通过GPIO48控制通断)
      列MUX的COM引脚通过10kΩ下拉电阻接地, 同时接ESP32-S3 GPIO4
```

#### 三、关于DC头电池 —— 购买指南

**为什么选DC头电池：**
- 充放电用**同一个DC口**，充电时只需插上充电器，即插即充
- DC头为通用的 **5.5×2.1mm** 规格，物料一致容易购买
- 不用自己焊接XT60或T插等航模插头，新手友好

**购买要点：**

```
✅ 搜索关键词: "12V锂电池 DC5.5 2.1 5000mAh"
✅ 电压: 标称 12V (实际 11.1V~12.6V, 即3串锂电)  
✅ 容量: ≥ 5000mAh (推荐5000-6000mAh, 续航约2小时)
✅ 输出接口: DC 5.5×2.1mm 公头 (最常见的DC头规格)
✅ 带保护板: 过充/过放/过流/短路保护 (一般都自带)
✅ 充电器: 12.6V 2A DC头充电器 (很多电池套餐含充电器)

❌ 避免: 没有保护板的裸电芯
❌ 避免: XT60/T插头的航模电池 (需要额外转接)
❌ 避免: 6V/7.4V 的2串电池 (电压不够驱动12V推杆和电机)
```

**典型电池外观：** 类似黑色塑料壳方块，一头有DC圆口，标签写着"12V 5000mAh 18650"或"12V聚合物锂电池"。

#### 四、供电接线步骤 (⚡ 按顺序操作)

> ⚠️ **安全第一**：接线时**务必断开电池**！不要带电操作！

**Step 1: LM2596先调压，再接设备**

```
工具: 万用表 + 小一字螺丝刀

1. 将LM2596的IN+/IN-接到12V电源 (暂用电池或12V适配器)
2. 用万用表测量 OUT+/OUT- 输出电压
3. 用小螺丝刀旋转板上蓝色电位器:
   - 逆时针 → 电压降低
   - 顺时针 → 电压升高
4. 精确调到 5.10V ± 0.05V
5. 两个LM2596都要调！分别标记为 #1(树莓派) 和 #2(ESP32)

⚠️ 严禁未调压就接树莓派！出厂可能是12V输出，会烧板子！
```

**Step 2: 制作DC母座转接**

```
DC母座转接板 (螺丝端子型):
  (+) 正极螺丝  ──→ 船型开关一端
  (-) 负极螺丝  ──→ 公共GND总线

使用方法: 电池DC公头插入此母座即可供电
         充电时拔掉电池, 充电器DC头直接插电池的DC口
```

**Step 3: 船型开关接入**

```
船型开关 (两脚):
  脚1 ← DC母座正极
  脚2 → 12V正极总线 (接到所有用电设备的正极)

作用: 按下 = 通电, 弹起 = 断电
安装: 用热熔胶固定在底盘侧面, 方便随时开关
```

**Step 4: 12V总线分配**

```
船型开关输出 (12V正极总线):
  ├──→ L298N #1 的12V输入端 (供前轮电机)
  ├──→ L298N #2 的12V输入端 (供后轮电机 + 12V电动推杆)
  ├──→ LM2596 #1 IN+ (降压给树莓派)
  └──→ LM2596 #2 IN+ (降压给ESP32)

DC母座负极 (GND总线):
  ├──→ L298N #1 GND
  ├──→ L298N #2 GND
  ├──→ LM2596 #1 IN-
  ├──→ LM2596 #2 IN-
  └──→ 面包板GND轨

💡 建议用接线端子排汇总12V和GND, 比直接拧线更可靠
```

**Step 5: LM2596 → 树莓派4B**

```
方案A: USB-C供电线 (推荐)
  LM2596 #1 OUT+ (5V)  ──→ USB-C供电线 红线 (+)
  LM2596 #1 OUT- (GND) ──→ USB-C供电线 黑线 (-)
  USB-C头插入树莓派USB-C供电口

方案B: GPIO供电 (高级用户)
  LM2596 #1 OUT+ (5V)  ──→ 树莓派 GPIO Pin 2 或 Pin 4 (5V)
  LM2596 #1 OUT- (GND) ──→ 树莓派 GPIO Pin 6 (GND)
  ⚠️ 此方案绕过了保险丝和电压检测，需确保5V精确！

树莓派4B满载功耗约 5V × 2.5A = 12.5W
LM2596 最大输出 3A, 足够使用
```

**Step 6: LM2596 → ESP32-S3 + 传感器**

```
LM2596 #2 OUT+ (5V)  ──→ 面包板正极轨
LM2596 #2 OUT- (GND) ──→ 面包板负极轨

ESP32-S3:
  VIN(5V)   ← 面包板正极轨 (通过板载LDO降为3.3V)
  GND       ← 面包板负极轨

各传感器:
  VCC(3.3V) ← ESP32-S3 3V3引脚 (或面包板3.3V, 取决于传感器电压要求)
  GND       ← 面包板负极轨

⚠️ MAX30102 / MLX90614 / MPU6050 都是 3.3V 器件!
   接到 ESP32-S3 的 3V3 输出, 不要直接接5V!
```

**Step 7: 电池电压检测 (可选但推荐)**

```
12V → 100kΩ电阻 → 中间节点 → 33kΩ电阻 → GND
                      ↓
               ESP32-S3 GPIO6 (ADC1_CH5)

分压比: 33/(100+33) = 0.248
满电12.6V → ADC读到 3.12V (ESP32安全范围 0-3.3V)
欠压9.9V  → ADC读到 2.45V

固件已有检测代码, 每30秒上报一次电池电量
```

| L298N×3 | 12V | 自耗 ~75mA | 0.9W | 驱动板静态 |
| LM2596×2 | 12V→5V | 转换损耗 ~10% | ~2W | 降压损耗 |
| LCD/功放/其他 | 5V/3.3V | ~300mA | ~1.5W | 外设合计 |
| **合计 (行驶)** | | | **~35W** | |
| **合计 (静止监测)** | | | **~15W** | 电机不转时 |

```
续航估算 (5000mAh / 12V = 60Wh 电池):
  行驶模式: 60Wh ÷ 35W ≈ 1.7小时
  静止监测: 60Wh ÷ 15W ≈ 4.0小时
  混合使用: 约 2~3小时 (足够Demo演示)
```

#### 六、充电方法

```
1. 关闭船型开关 (断开所有负载)
2. 将12.6V DC充电器插入电池的DC接口
   - 电池和充电器用的是同一个DC 5.5×2.1mm口
   - 充电器插头插入电池的DC口即可充电
3. 充电指示灯:
   红灯 → 充电中 (约2-3小时充满5000mAh)
   绿灯 → 充满 (自动停充)
4. 充满后拔掉充电器

⚠️ 注意事项:
  - 充电时不要同时使用设备 (虽然保护板支持, 但不推荐)
  - 使用配套12.6V充电器, 不要用12V的
  - 长期不用请每月补充电一次, 防止过放
  - 充电环境温度 0~40℃
```

#### 七、安全注意事项

```
🔴 绝对禁止:
  × 短路电池正负极 (会导致起火!)
  × 未调压就接树莓派/ESP32 (会烧坏!)
  × 使用非配套充电器充电
  × 在密封环境中充电
  × 电池鼓包后继续使用

🟡 特别注意:
  △ 接线前务必断开电池 (拔掉DC头)
  △ LM2596必须先用万用表确认5V后再连接设备
  △ 所有GND必须共地 (电池GND、L298N GND、面包板GND互连)
  △ 12V线建议用18AWG或更粗的线材
  △ 5V线至少22AWG (尤其是树莓派供电线)

🟢 推荐做法:
  ✓ 接线端子排汇总12V/GND, 整洁可靠
  ✓ 船型开关方便随时断电
  ✓ 加装电池电压显示器 (3S锂电电量显示)
  ✓ 各模块供电线加装保险丝 (2A给树莓派, 5A给电机)
  ✓ 用热缩管保护裸露焊点
```

#### 八、电源故障排查

| 故障现象 | 可能原因 | 解决方法 |
|---------|---------|---------|
| 全部不工作 | 电池没电/船型开关没开 | 检查电池电压、开关状态 |
| 树莓派不启动 | LM2596输出不是5V | 万用表测LM2596 #1输出 |
| 树莓派闪电符号⚡ | 供电不足(<4.65V) | 检查USB-C线和LM2596 |
| ESP32反复重启 | 供电不稳/线太细 | 加装100μF电容到5V/GND |
| 电机转但无力 | 电池电压低 | 充电；检查电池电压>10V |
| 推杆不动 | 未接12V或L298N使能 | 检查L298N #2的12V和EN |
| LCD不亮 | 树莓派供电不足 | LCD和树莓派分开供电 |
| 充电指示灯不亮 | 充电器没插好/DC头接触差 | 重新插拔、清洁DC接口 |
