# 智能护理病床 ESP32 接线指南

## 📋 物料清单

- ESP32-S3 开发板 ×1
- CD4051BE 8选1多路选择器 (紫色板) ×2
- HK-2000R+ 脉搏传感器 (带3.5mm插头) ×1
- Velostat 8×8 压力矩阵 ×1
- MLX90614 红外测温模块 ×1
- MPU6050/9250 姿态传感器 ×1
- MAX30102 心率血氧模块 ×1
- 杜邦线若干

---

## 🔌 接线方案

### 1. I2C 总线传感器 (共用 SDA/SCL)

所有 I2C 传感器并联在同一总线上:

```
┌─────────────────┬──────────┬──────────┬──────────┐
│ 传感器          │ VCC      │ GND      │ 其他     │
├─────────────────┼──────────┼──────────┼──────────┤
│ MLX90614        │ 3.3V     │ GND      │          │
│ MPU6050/9250    │ 3.3V     │ GND      │          │
│ MAX30102        │ 3.3V     │ GND      │          │
└─────────────────┴──────────┴──────────┴──────────┘

公共 I2C 引脚:
  SDA → ESP32 GPIO2
  SCL → ESP32 GPIO1
```

**注意:** 这3个传感器的 SDA/SCL 全部连到 GPIO2/GPIO1,不需要分开。

### 2. HK-2000R+ 脉搏传感器

#### 3.5mm TRS 插头线色识别:

拆开插头外壳,会看到三根线焊在插头上:

```
       Tip (尖端)
        │
   ┌────┴────┐
   │    /    │  Ring (环)
   │   /     │   │
   └──/──────┘   │
     /           │
    /            │  Sleeve (套筒)
   /             │   │
  ─────────────  ─  ───
   黄线        红线  黑线
  (信号)      (VCC) (GND)
```

#### 接线表:

```
┌─────────────────┬──────────────┐
│ 3.5mm 插头      │ ESP32        │
├─────────────────┼──────────────┤
│ 黑线 (Sleeve)   │ GND          │
│ 红线 (Ring)     │ 3.3V         │
│ 黄线 (Tip)      │ GPIO5 (ADC)  │
└─────────────────┴──────────────┘
```

### 3. CD4051 多路选择器 (压力矩阵)

需要 **2片** CD4051:
- **MUX1** (行选择) - 驱动 Velostat 的 8 行
- **MUX2** (列选择) - 读取 Velostat 的 8 列

#### CD4051 引脚定义 (16脚 DIP 封装):

```
正面看,缺口朝上:

     ┌─────U─────┐
  Y6 │1        16│ VDD (3.3V)
  Y4 │2        15│ Y0
  Y7 │3        14│ Y1
  Y5 │4        13│ Y2
 INH │5        12│ Y3
 VEE │6        11│ COM ← 公共端(重要!)
 GND │7        10│ S2 (C)
S0(A)│8         9│ S1 (B)
     └───────────┘
```

#### MUX1 (行选择) 接线表:

```
┌─────────────┬──────────────┬─────────────────┐
│ CD4051 引脚 │ ESP32        │ 说明            │
├─────────────┼──────────────┼─────────────────┤
│ 16 (VDD)    │ 3.3V         │ 电源+           │
│ 7 (GND)     │ GND          │ 电源-           │
│ 6 (VEE)     │ GND          │ 负电压(接地)    │
│ 5 (INH)     │ GND          │ 使能(低电平有效)│
│             │              │                 │
│ 8 (S0/A)    │ GPIO39       │ 地址位0         │
│ 9 (S1/B)    │ GPIO40       │ 地址位1         │
│ 10 (S2/C)   │ GPIO41       │ 地址位2         │
│             │              │                 │
│ 11 (COM)    │ GPIO48       │ 行驱动信号      │
│             │              │                 │
│ Y0-Y7       │ Velostat行0-7│ 连到压力矩阵行  │
│ (1,2,3,4,   │              │                 │
│  12,13,14,  │              │                 │
│  15)        │              │                 │
└─────────────┴──────────────┴─────────────────┘
```

#### MUX2 (列选择) 接线表:

```
┌─────────────┬──────────────┬─────────────────┐
│ CD4051 引脚 │ ESP32        │ 说明            │
├─────────────┼──────────────┼─────────────────┤
│ 16 (VDD)    │ 3.3V         │ 电源+           │
│ 7 (GND)     │ GND          │ 电源-           │
│ 6 (VEE)     │ GND          │ 负电压(接地)    │
│ 5 (INH)     │ GND          │ 使能(低电平有效)│
│             │              │                 │
│ 8 (S0/A)    │ GPIO42       │ 地址位0         │
│ 9 (S1/B)    │ GPIO21       │ 地址位1         │
│ 10 (S2/C)   │ GPIO47       │ 地址位2         │
│             │              │                 │
│ 11 (COM)    │ GPIO4 (ADC)  │ 读取压力值      │
│             │              │                 │
│ Y0-Y7       │ Velostat列0-7│ 连到压力矩阵列  │
│ (1,2,3,4,   │              │                 │
│  12,13,14,  │              │                 │
│  15)        │              │                 │
└─────────────┴──────────────┴─────────────────┘
```

#### Velostat 压力矩阵连接:

```
MUX1 的 Y0-Y7 → Velostat 行引脚 0-7
MUX2 的 Y0-Y7 → Velostat 列引脚 0-7
```

工作原理:
1. 通过 S0/S1/S2 选择 MUX1 的某一行 (0-7)
2. GPIO48 输出 HIGH,给选中的行加电压
3. 通过 S0/S1/S2 选择 MUX2 的某一列 (0-7)
4. GPIO4 (ADC) 读取该交叉点的压力值
5. 扫描 8×8 = 64 个点

### 4. 电池电压检测 (可选)

如果使用 12V 锂电池供电,需要分压电路:

```
12V 电池+
   │
   ├─── 100kΩ 电阻 ───┬─── GPIO6 (ADC)
   │                  │
   │                 33kΩ 电阻
   │                  │
GND ─────────────────┴─── GND
```

分压后电压: `12V × 33k/(100k+33k) = 2.97V` (ESP32 ADC 安全范围)

## 📌 引脚占用汇总

```
┌──────────┬─────────────────────────────┐
│ GPIO     │ 功能                        │
├──────────┼─────────────────────────────┤
│ GPIO1    │ I2C SCL (3个传感器共用)     │
│ GPIO2    │ I2C SDA (3个传感器共用)     │
│ GPIO4    │ 压力矩阵列 ADC 读取         │
│ GPIO5    │ 脉搏传感器 ADC              │
│ GPIO6    │ 电池电压 ADC (可选)         │
│ GPIO21   │ 列MUX S1                    │
│ GPIO39   │ 行MUX S0                    │
│ GPIO40   │ 行MUX S1                    │
│ GPIO41   │ 行MUX S2                    │
│ GPIO42   │ 列MUX S0                    │
│ GPIO47   │ 列MUX S2                    │
│ GPIO48   │ 行MUX 驱动信号              │
└──────────┴─────────────────────────────┘
```

## ⚠️ 注意事项

1. **所有传感器必须用 3.3V 供电**,ESP32-S3 不是 5V 容忍的
2. **I2C 上拉电阻:** 大部分模块自带 4.7kΩ 上拉,如果不工作可能需要外加
3. **ADC 引脚:** ESP32-S3 的 ADC2 在 WiFi 开启时不可用,本项目全部用 ADC1
4. **线长控制:** I2C 总线杜邦线建议 < 20cm,过长会导致通信不稳定
5. **CD4051 的 INH 引脚必须接 GND**,否则芯片不工作
6. **CD4051 的 VEE 引脚接 GND** (只处理正电压,不需要负电源)

## 🧪 测试步骤

### 1. 先测 I2C 传感器

上传 `main_i2c_scan.cpp`,应该扫描到:
- 0x57 → MAX30102
- 0x5A → MLX90614
- 0x68 或 0x38 → MPU6050/9250

### 2. 再测脉搏传感器

手指按住传感器,串口监视器应该看到 ADC 值波动 (心跳节奏)

### 3. 最后测压力矩阵

按压 Velostat 矩阵任意点,MQTT 应该发布对应位置的压力值

## 🐛 常见问题

### Q1: MLX90614 初始化失败?
→ 确保它在 MAX30102 之前初始化 (已在代码中修复)

### Q2: MPU6050 WHO_AM_I 不是 0x68?
→ 可能是 MPU6500/9250,代码已兼容 0x38

### Q3: 压力矩阵所有值都是 0?
→ 检查 CD4051 的 INH 引脚是否接地,VDD 是否接 3.3V

### Q4: 脉搏传感器读数不变?
→ 确认黄线接到 GPIO5,红线接 3.3V (不是 5V)
